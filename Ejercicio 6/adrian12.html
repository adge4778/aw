Software libre



Escritorio

7-zip

7-zip es un compresor/descompresor de archivos que maneja los formatos ZIP, RAR, CAB, GZIP, BZIP2 y TAR, así como un formato propio, 7z

Audacity

Audacity es un editor de audio. Permite reproducir, grabar, editar, mezclar, aplicar efectos, etc.

GIMP

GIMP (GNU Image Manipulation Program) es un programa de retoque fotográfico y composición y edición de imágenes

Inkscape

Inkscape es un editor de gráficos vectoriales que utiliza el formato SVG y que también importa formatos como Postscript, EPS, JPEG, PNG, y TIFF y exporta en PNG y varios formatos vectoriales

LibreOffice

LibreOffice es una suite ofimática completa compuesta de procesador de textos, hoja de cálculo, presentaciones, programa de dibujo, base de datos y
editor de ecuaciones.

Internet

FileZilla

FileZilla es un cliente de FTP

Firefox

Firefox es un navegador web altamente personalizable y ampliable mediante los cientos de extensiones y temas disponible

HTTrack

HTTrack permite descargar sitios webs completos para su consulta offline

Pidgin

Pidgen es un cliente de mensajeria instantanea compatible con las redes AIM (protocolos Oscar y TOC). ICQ, MSN Meseenger, Yahoo, IRC, Jabber, Gadu-Gadu, Zephyr y otras.

Thunderbird

Thunderbird es un cliente de correo de los creadores de Firefox. Incorpora multitud de caracteristicas avanzadas y, como Firefox, puede personalizarse mediante temas y ampliar sus posibilidades mediante extensiones.
